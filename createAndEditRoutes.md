# Размещение и редактирование дорожек

<div style="text-align:justify;">
<p>ARES предоставляет множество способов для ручного размещения и редактирования дорожек и переходных отверстий. В частности, существующие дорожки можно отредактировать любым из следующих методов:</p>
<ul>
<li>с помощью размещения – разместите новый участок дорожки поверх старого и новый участок заменит старый. Это позволяет легко изменять толщину участков дорожки.</li>
<li>перетаскиванием участка – траектория выделенной дорожки может быть легко изменена, перемещая участки. Соседние сегменты автоматически растягиваются или сужаются, сохраняя ортогональность дорожки.</li>
</ul>
<p>В любом случае, ARES проверяет новые или модифицированные участки дорожки с учетом данных о связях, приведенных в списке соединений и физических правилах проектирования для печатной платы. Участки дорожки, которые нарушают правила связи, мигают желтым цветом, обозначая, что они разведены неверно, а дорожки, нарушающие правила проектирования, заносятся в список, доступный через диалоговое окно программы проверки правил в нижней правой части программы.</p>
<center><strong>Разводка дорожек без загруженного списка соединений</strong></center>
<p>Размещение дорожки на выбранном слое осуществляется путем выбора иконки <strong>Track Mode</strong> в левой панели инструментов и затем нажав левую кнопку мыши в каждой точке ломанной вдоль требуемой траектории. Для завершения проведения дорожки нажмите правую кнопку мыши. Заметьте, что курсор мыши изменится (примет форму карандаша), когда пользователь находится в режиме размещения дорожки.</p>
<p>Двойной щелчок в одном и том же месте приведет к размещению переходного отверстия и выбору следующего слоя из последовательности пар слоев. Тип используемого переходного отверстия можно изменить, выбрав иконку <strong>Via Mode</strong> и один из стилей переходных отверстий из селектора объектов. Пользователь также можете разместить, заменить, выделить, переместить и удалить переходные отверстия вручную в этом режиме.</p>
<p>Разработчик можете изменить слои во время проведения дорожек, используя «горячие» клавиши для работы с селектором слоев. Клавишами по умолчанию являются Page Up, Page Down, CTRL+Page Up, CTRL+Page Down. «Горячие» клавиши можете изменить с помощью команды <strong>Set Keyboard Mapping</strong> из меню <strong>System</strong>.</p>
<p>После того как пользователь начал разводить дорожку, для отмены проведения дорожки необходимо нажать клавишу ESC.</p>
